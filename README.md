# ngFx Demo

> An animation demo using angular.js and the ngFx module

## Getting Started

```sh
git clone git@github.com:jasonshark/ngfx-demo.git
cd angular-starter
npm install -g http-server
http-server
```

This starts the website on **http://localhost:8080**
